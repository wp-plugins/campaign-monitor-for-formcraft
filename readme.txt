== Description ==

This add-on allows you to add subscribers to your Campaign Monitor lists, with forms made with [FormCraft](http://formcraft-wp.com/). You can read the tutorial [here](http://formcraft-wp.com/help/how-to-configure-campaign-monitor-add-on/)
